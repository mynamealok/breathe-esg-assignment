# Research Sources

## SAP

Researched:
- SAP reporting exports
- CSV based reporting workflows
- OData services

Learned:
SAP data often contains inconsistent field names, units and date formats.

Sample Data:
Qty, Unit

Potential Production Issues:
- Localization
- Missing lookup tables
- Unit inconsistency

---

## Utility Data

Researched:
Utility portal CSV exports

Learned:
Data commonly includes consumption, billing period and meter information.

Sample Data:
kWh values

Potential Production Issues:
- Missing periods
- Meter changes
- Tariff complexity

## Corporate Travel

Researched:
Concur style exports

Learned:
Travel records may include flights, hotels and ground transport.

Sample Data:
TravelType, DistanceKM

Potential Production Issues:
- Missing distances
- Airport code conversions
- Multi-leg journeys
